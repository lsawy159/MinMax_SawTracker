# SAWTRACKER Design System

هذا المستند يعرّف التوكنز الأساسية للواجهة بعد بداية مرحلة Foundation.

## 1) Color Tokens

### Brand
- `brand.gold`: `#FECE14`
- `brand.hover`: `#EAB308`
- `brand.subtle`: `#FEF9C3`

### Neutral
- `neutral.950`: `#0A0A0A`
- `neutral.900`: `#171717`
- `neutral.700`: `#404040`
- `neutral.500`: `#737373`
- `neutral.300`: `#D4D4D4`
- `neutral.200`: `#E5E5E5`
- `neutral.100`: `#F5F5F5`
- `neutral.50`: `#FAFAFA`

### Semantic
- `success`: `#10B981` / subtle `#ECFDF5`
- `warning`: `#F59E0B` / subtle `#FFFBEB`
- `danger`: `#EF4444` / subtle `#FEF2F2`
- `info`: `#3B82F6` / subtle `#EFF6FF`

## 2) Typography

### Font Families
- `sans`: `IBM Plex Sans Arabic`, `Inter`
- `display`: `IBM Plex Sans Arabic`, `Inter`
- `mono`: `IBM Plex Mono`

### Type Scale
- Display: `32 / 40` - weight `700`
- Heading-1: `24 / 32` - weight `700`
- Heading-2: `20 / 28` - weight `600`
- Heading-3: `18 / 26` - weight `600`
- Body-lg: `16 / 24` - weight `400`
- Body: `14 / 20` - weight `400`
- Body-sm: `13 / 18` - weight `400`
- Caption: `12 / 16` - weight `500`

## 3) Spacing Scale

- `4px`, `8px`, `12px`, `16px`, `24px`, `32px`, `48px`, `64px`

## 4) Radius Scale

- `sm`: `6px`
- `md`: `8px`
- `lg`: `12px`
- `xl`: `16px`
- `full`: `9999px`

## 5) Elevation

- `shadow-xs`: `0 1px 2px rgba(0,0,0,0.04)`
- `shadow-sm`: `0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04)`
- `shadow-md`: `0 4px 12px rgba(0,0,0,0.08)`
- `shadow-lg`: `0 12px 32px rgba(0,0,0,0.10)`
- `shadow-xl`: `0 24px 48px rgba(0,0,0,0.14)`
- `shadow-focus`: `0 0 0 3px rgba(254, 206, 20, 0.30)`

## 6) Motion

- Duration: `150ms`, `200ms`, `300ms`
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)`
