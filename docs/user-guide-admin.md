## الفصل 1: الدخول والإعدادات الأساسية

### الدخول للمنصة

1. اذهب إلى https://sawtracker.com
2. أدخل بريدك الإلكتروني
3. أدخل كلمة المرور (حد أدنى 8 أحرف)
4. اضغط "دخول"

### تغيير كلمة المرور

1. اذهب إلى **الإعدادات** (أيقونة الترس)
2. اختر **الأمان**
3. اضغط **تغيير كلمة المرور**
4. أدخل الكلمة الجديدة (قوية: أحرف كبيرة + صغيرة + أرقام + رموز)

### الملف الشخصي

1. اضغط صورتك الشخصية (أعلى يمين)
2. اختر **ملفي**
3. حدّث:
   - الاسم الكامل
   - البريد الإلكتروني
   - الصورة الشخصية

## الفصل 2: إدارة الشركات

### إضافة شركة جديدة

1. اذهب إلى **الشركات**
2. اضغط **+ شركة جديدة**
3. أدخل:
   - **الاسم**: اسم الشركة
   - **المدينة**: موقع الشركة
   - **الهاتف**: رقم الاتصال
   - **البريد الإلكتروني**: البريد الرسمي
4. اضغط **حفظ**

### عرض الشركات

في صفحة **الشركات**:
- رؤية قائمة بجميع الشركات
- البحث بـ الاسم
- تصفية حسب المدينة
- الضغط على اسم الشركة لعرض التفاصيل

### تعديل معلومات الشركة

1. اذهب إلى **الشركات**
2. اضغط على الشركة
3. اضغط **تعديل**
4. غيّر البيانات المطلوبة
5. اضغط **حفظ**

### حذف شركة

⚠️ **تحذير**: الحذف نهائي ولا يمكن استرجاع البيانات

1. اضغط على الشركة
2. اضغط **حذف**
3. أكّد الحذف

## الفصل 3: إدارة الموظفين

### إضافة موظف

1. اذهب إلى **الموظفين**
2. اضغط **+ موظف جديد**
3. أدخل:
   - **الاسم**: الأول والأخير
   - **البريد الإلكتروني**: البريد الشركة
   - **الهاتف**: رقم الاتصال
   - **الراتب الأساسي**: الراتب الشهري
   - **الدور**: admin / accountant / data-entry / user
4. اضغط **إنشاء حساب**

### تحديث بيانات الموظف

1. اذهب إلى **الموظفين**
2. ابحث عن الموظف
3. اضغط على اسمه
4. اضغط **تعديل**
5. حدّث البيانات
6. اضغط **حفظ**

### تعيين الأدوار والصلاحيات

#### الأدوار المتاحة:

- **admin**: وصول كامل (إضافة/تعديل/حذف)
- **accountant**: معالجة الرواتب والفواتير
- **data-entry**: إدخال البيانات فقط
- **user**: عرض البيانات فقط

#### تعيين دور:

1. اذهب إلى **الصلاحيات**
2. ابحث عن الموظف
3. اختر الدور من القائمة المنسدلة
4. اضغط **حفظ**

### تعطيل حساب موظف

عند تركه العمل:

1. اذهب إلى **الموظفين**
2. اضغط على الموظف
3. اضغط **تعطيل الحساب**
4. الحساب لن يقدر يدخل لكن البيانات محفوظة

## الفصل 4: معالجة الرواتب

### عرض ملخص الرواتب

1. اذهب إلى **الرواتب**
2. ستشوف:
   - عدد مسيرات الرواتب
   - إجمالي الرواتب المدفوعة
   - آخر معالجة رواتب

### إنشاء مسيرة رواتب جديدة

1. اضغط **+ مسيرة جديدة**
2. اختر الشهر والسنة
3. اضغط **إنشاء**
4. تحقق من الموظفين والمبالغ
5. اضغط **معالجة**

### عرض تفاصيل الراتب

1. اختر مسيرة الرواتب
2. اضغط على اسم الموظف
3. شوف:
   - الراتب الأساسي
   - الاستقطاعات
   - الإجمالي النهائي

---

## الفصل 5: التقارير

### تقرير الموظفين

1. اذهب إلى **التقارير**
2. اختر **تقرير الموظفين**
3. اختر الشهر (اختياري)
4. اضغط **تحميل PDF**

**المعلومات**:
- عدد الموظفين النشطين
- قائمة بالموظفين
- إجمالي الراتب

### تقرير الرواتب

1. اختر **تقرير الرواتب**
2. اختر الشهر
3. اضغط **عرض**

**المعلومات**:
- إجمالي الراتب المدفوع
- عدد الموظفين المدفوعين
- متوسط الراتب

### تقرير الاستقطاعات

1. اختر **تقرير الاستقطاعات**
2. اختر الفترة (يومي/أسبوعي/شهري)
3. عرض:
   - أنواع الاستقطاعات
   - المبالغ
   - الموظفين المتأثرين

## الفصل 6: الإعدادات الإدارية

### النسخ الاحتياطية

1. اذهب إلى **الإعدادات** → **النسخ الاحتياطية**
2. اضغط **عمل نسخة احتياطية الآن**
3. انتظر حتى تكتمل (عادة بأقل من دقيقة)

### الاستعادة من نسخة احتياطية

⚠️ **تحذير**: الاستعادة ستحذف البيانات الحالية

1. اذهب إلى **الإعدادات** → **النسخ الاحتياطية**
2. اختر التاريخ المطلوب
3. اضغط **استعادة**
4. أكّد العملية

### إدارة المستخدمين

1. اذهب إلى **الإعدادات** → **المستخدمين**
2. شوف قائمة بجميع المستخدمين
3. يمكنك:
   - **تعديل**: تغيير الدور
   - **تعطيل**: منع الدخول
   - **حذف**: حذف نهائي

### الأمان والخصوصية

1. اذهب إلى **الإعدادات** → **الأمان**
2. شوف:
   - كلمة المرور الحالية
   - آخر تسجيل دخول
   - الأجهزة النشطة

## الفصل 7: استكشاف الأخطاء

### المنصة بطيئة

1. حاول تحديث الصفحة (F5)
2. امسح ذاكرة التخزين (Ctrl+Shift+Delete)
3. جرّب متصفح آخر
4. تواصل مع الدعم إذا استمرت المشكلة

### لا أقدر أدخل

1. تأكد من البريد الإلكتروني صحيح
2. تأكد من كلمة المرور صحيحة
3. اضغط **نسيت كلمة المرور** عشان تعيدها
4. امسح ملفات تعريف الارتباط وحاول مجدداً

### البيانات لا تظهر

1. حاول تحديث الصفحة
2. تأكد من اتصال الإنترنت
3. فتّش شاشة الأخطاء (أعلى الصفحة)
4. لو استمرت المشكلة، تواصل مع الدعم
