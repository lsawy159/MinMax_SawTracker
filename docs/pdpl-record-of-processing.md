## 1. نشاطات معالجة البيانات

### 1.1 بيانات الموظفين

| البند | التفاصيل |
|-------|----------|
| **الغرض** | إدارة شؤون الموظفين، متابعة تصاريح الإقامة، الرواتب، المشاريع |
| **فئات البيانات** | الاسم، رقم الهوية/الإقامة، رقم الجواز، تاريخ انتهاء الإقامة، تاريخ انتهاء التأمين الصحي، الراتب، بيانات الاستقطاعات |
| **الأطراف المعنية** | موظفو الشركات المسجلة في النظام |
| **القاعدة القانونية** | التزام قانوني (نظام العمل السعودي)، عقد العمل |
| **مدة الاحتفاظ** | 5 سنوات بعد إنهاء الخدمة، وفق نظام العمل |
| **مشاركة البيانات** | لا تُشارك مع أطراف خارجية إلا عند الضرورة القانونية |
| **التخزين** | Supabase PostgreSQL (منطقة: EU-West) |

### 1.2 بيانات المستخدمين (مديري النظام)

| البند | التفاصيل |
|-------|----------|
| **الغرض** | تسجيل الدخول، التحكم في الصلاحيات، تدقيق العمليات |
| **فئات البيانات** | اسم المستخدم، البريد الإلكتروني، دور المستخدم، سجل الجلسات، سجل التدقيق |
| **الأطراف المعنية** | موظفو الإدارة المخولون باستخدام النظام |
| **القاعدة القانونية** | عقد العمل، المصلحة المشروعة (أمن النظام) |
| **مدة الاحتفاظ** | طوال فترة العمل + 2 سنة للسجلات التدقيقية |
| **مشاركة البيانات** | لا تُشارك |
| **التخزين** | Supabase PostgreSQL (منطقة: EU-West) |

### 1.3 بيانات بريد إلكتروني (إشعارات)

| البند | التفاصيل |
|-------|----------|
| **الغرض** | إرسال تنبيهات انتهاء الإقامة والتأمين، تقارير يومية |
| **فئات البيانات** | عنوان البريد الإلكتروني، محتوى الرسالة |
| **الأطراف المعنية** | مستلمو الإشعارات المحددون في الإعدادات |
| **القاعدة القانونية** | المصلحة المشروعة (تشغيل الأعمال) |
| **مدة الاحتفاظ** | 90 يوماً (سجل قائمة الانتظار) |
| **مشاركة البيانات** | Resend.com (مزود خدمة البريد) — اتفاقية معالجة بيانات قائمة |
| **التخزين** | Supabase PostgreSQL + Resend |

## 2. حقوق أصحاب البيانات

| الحق | الإجراء |
|------|---------|
| الاطلاع | يتقدم بطلب كتابي للمسؤول عبر البريد الإلكتروني |
| التصحيح | يُقدَّم الطلب للمسؤول → يُعدَّل الإدخال في النظام |
| المحو | يُقيَّم بناءً على الالتزامات القانونية (نظام العمل) |
| النقل | بيانات الموظف متاحة بصيغة Excel عند الطلب |
| الاعتراض | يُرفع للمسؤول المعين |

---

## 3. إجراءات الأمن المطبقة

| الإجراء | الوصف |
|---------|-------|
| **التشفير أثناء النقل** | TLS 1.2+ على كافة الاتصالات |
| **التشفير في الراحة** | AES-256 (Supabase) + AES-256-GCM للنسخ الاحتياطية |
| **التحكم في الوصول** | RLS (Row Level Security) على مستوى قاعدة البيانات |
| **الصلاحيات** | نظام أدوار متعدد المستويات (admin/accountant/data-entry) |
| **سجل التدقيق** | جميع عمليات القراءة/الكتابة مسجلة في audit_log |
| **النسخ الاحتياطي** | يومي، مشفر، محتفظ به 30 يوماً |
| **إدارة الجلسات** | JWT قصير الأجل، مهلة خمول 30 دقيقة |
